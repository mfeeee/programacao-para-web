HTML

O que é?

HTML (acrônimo para a expressão inglesa HyperText Markup Language, que significa Linguagem de Marcação de Hipertexto) é uma linguagem de marcação utilizada para produzir páginas na Web. Documentos HTML podem ser interpretados por navegadores.

Sobre o curso:
Este curso ensina desde os conceitos básicos da linguagem HTML, até a criação de formulários para a Web, ideal para iniciantes, o par quem quer um guia de consulta, tam´bem possui um Glossário com todas as palavras mais utilizadas na internet

Dreamweaver

O que é?

O Macromedia Dreamweaver é um software de criação para web criada pela Macromedia (agora Adobe Systems), que está atualmente na versão CS3. Versões iniciais da aplicação serviam como um simples editor HTML WYSIWYG ("What You See Is What You Get")
